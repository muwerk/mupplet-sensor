Detailed Description

mup_i2c_registers.h implements the i2c register read/write protocols used by many sensors

The I2CRegisters class takes a pointer to a TwoWire object and an i2c address and provides a set of functions to read and write registers on the i2c bus.

This module relies only on the Arudino/ESP Wire library and can be reused by any Arduino project.
