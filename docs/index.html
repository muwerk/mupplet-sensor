mupplet-sensor is a collection of hardware sensor applets for the muwerk scheduler

Introduction

mupplet-sensor implements the following classes based on the cooperative scheduler muwerk:

  • ustd::BinarySensor
  • ustd::AnalogSensor
  • ustd::IlluminanceLdr
  • ustd::IlluminanceTSL2561
  • ustd::PressTempBMP180
  • ustd::PressTempBMP280
  • ustd::PressTempHumBME280
  • ustd::TempHumDHT
  • ustd::GammaGDK101
  • ustd::RainAD
  • ustd::PowerBl0937

Helper:

  • ustd::I2CRegisters

For an overview, see: mupplet-sensor readme

Libraries are header-only and should work with any c++11 compiler and and support platforms esp8266 and esp32.

This library requires the libraries ustd, muwerk and requires a platform define.
